## STEP 1 — Set up Supabase (do this FIRST, takes 2 min)

1. Go to https://supabase.com → Sign in
2. Open your project (tldbjsqhjqapdwmfeuwq)
3. Click "SQL Editor" in the left sidebar → "New Query"
4. Copy-paste the entire contents of `supabase_setup.sql`
5. Click "Run"
6. You should see: "Success. No rows returned"
7. That's it. Your `users` and `orders` tables are now ready.

## HOW THE NEW FLOW WORKS

### New user journey:
1. User clicks "Get Started" on any page
2. Goes to `subscribe.html` — SEES PLANS IMMEDIATELY (no login wall)
3. Clicks "Subscribe Now" on any plan
4. If not logged in → redirected to `register.html` with plan pre-selected
5. Fills name, email, phone, password → clicks "Create Account"
6. Account saved to Supabase `users` table with `plan_status: pending`
7. Immediately redirected back to `subscribe.html` with all details in URL
8. Razorpay payment opens pre-filled with their name/email/phone
9. On payment success → Supabase `users` table updated to `plan_status: active`
10. Payment saved to `orders` table
11. WhatsApp notification sent to your number (918796158321)
12. Success modal shown

### Returning user:
1. Goes to `login.html`
2. Enters email + password
3. Password hashed (SHA-256) and matched against Supabase
4. Session stored in localStorage
5. Redirected to subscribe page or library

### Admin (on login.html):
1. Scroll to bottom → click tiny "Admin Access" link
2. Enter password: SATISH@1467
3. Full user dashboard appears:
   - Total users, joined today, paid members, total revenue
   - Full user table with plan/status
   - Export CSV button
   - Clear All button (use carefully!)
